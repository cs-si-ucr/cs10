Exercise 1

Write a program that outputs a menu that asks the user to pick an option between 1 - 6. If the option is 6 then the program should exit. Options 1 - 5 will be for the exercises. If the output is not in the range from 1 - 6 then the program should alert the user that the option is invalid and ask the user to enter another option.

Until you implement the other exercises simply do nothing if the user chooses 2 - 5. All of the exercises for this week will go in this file.

Example output:

Lab 05 Menu
1. Do nothing
2. Exercise 2
3. Exercise 3
4. Exercise 4
5. Exercise 5
6. Exit

Enter your choice: 1

Lab 05 Menu
1. Do nothing
2. Exercise 2
3. Exercise 3
4. Exercise 4
5. Exercise 5
6. Exit

Enter your choice: 7
Invalid choice, try again.
Enter your choice: 6

Goodbye!

Another example:

Lab 05 Menu
1. Do nothing
2. Exercise 2
3. Exercise 3
4. Exercise 4
5. Exercise 5
6. Exit

Enter your choice: 2

Executing exercise 2...
(exercise 2 runs here)

Lab 05 Menu
1. Do nothing
2. Exercise 2
3. Exercise 3
4. Exercise 4
5. Exercise 5
6. Exit

Enter your choice: 6

Goodbye!

Exercise 2

Example output:

bgb

gug

Note that the order does not have to be exactly the same as above as long as all permutations are there (this order is the easiest, though).

Bonus

Allow a string of 4 characters and output all permutations of that word with 4 words on each line.

Exercise 5

Ask the user for an integer and output the factorial of that integer. Do not use multiplication.

In case you are unfamiliar with what the factorial function does, here is a brief definition:

  1. The factorial of a number n, denoted n!, is given by n! == n * (n-1) * (n-2) * (n-3) * ... * 3 * 2 * 1
  2. The special case of n == 0 is defined as follows:
  3. 0! == 1

Example output:

Please enter an integer: 5
5! == 120

Bonus

If the user enters a negative or floating point number alert the user that their input is invalid and re-prompt the user.

Example output 1 for Bonus:

Please enter an integer: -1
Error: Invalid Input!
Please enter an integer: 0
1

Example output 2 for Bonus:

Please enter an integer: 1.4
Error: Invalid Input!
Please enter an integer: 3.0
6
